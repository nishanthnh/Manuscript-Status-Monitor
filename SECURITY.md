# Security and Privacy

Please do not publish private manuscript details.

Before sharing screenshots, code, issues, or examples, remove:

- Tracker UUIDs
- Full tracker URLs
- Email addresses
- Manuscript titles
- Manuscript numbers
- Author names
- Journal submission identifiers

Anyone with a public tracking URL may be able to view the status page. Treat the URL like private information.

This project is for public tracker pages that you are allowed to access. Do not use it to bypass login, CAPTCHA, two-factor authentication, or publisher access controls.
